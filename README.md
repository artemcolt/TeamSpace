# Team Space Desktop

Electron + React + TypeScript desktop app stub for Windows and Linux.

## Scripts

- `npm run dev` starts Vite and Electron for local development.
- `npm run typecheck` checks renderer and Electron TypeScript.
- `npm run build` builds Electron, renderer, and distributable packages.

## Structure

- `src/main` contains Electron main and preload code.
- `src/renderer` contains the React application.
