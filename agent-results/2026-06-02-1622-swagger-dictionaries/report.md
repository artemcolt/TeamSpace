# Swagger-группа CRUD справочников

## Что было сделано
- Добавлена отдельная Swagger-группа `CRUD справочников` для CRUD-ручек парков, транспортных средств, водителей и причин.
- CRUD-ручки справочников исключены из Swagger-группы `Основной API`.
- Ручки проверки доступности транспортных средств и водителей оставлены в Swagger-группе `Основной API`.

## Проверка
- Выполнена сборка backend с прогоном тестовой задачи Gradle.
- Проверена конфигурация Swagger: доступны группы `CRUD справочников` и `Основной API`.
- Проверен состав группы `CRUD справочников`: группа содержит только CRUD-пути справочников.
- Проверен состав группы `Основной API`: CRUD-пути справочников отсутствуют.

## Риски и заметки
- Для сборки backend требуется Java 25.
